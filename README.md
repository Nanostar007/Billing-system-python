## Billing-system-python
This is a desktop billing program made with Python and Tkinter.
You enter customer name, add items with name quantity and price, see receipt preview right away, save as PDF.
Features

Enter any items you want, no fixed list
Add or remove item rows with one click
Receipt preview updates when you type
Shows current date in dd/mm/yyyy format
Random bill number each time
Save bill to PDF with date, name and total
Clear All button resets everything
No phone number field, only name

## Credit

Requirements
Python 3.8 or newer
tkinter (included with Python)
reportlab (for PDF)
Install reportlab
pip install reportlab

## How to run

Download or clone this repository
Open terminal or command prompt in the folder
Install reportlab if not done yet
pip install reportlab
Start the program
python billing.py

## Usage
Type customer name
Add items: name, quantity, price
Click Show Bill to see preview
Click Save as PDF to choose file location
Click Clear All to start new bill
